# Demos

These pages exercise the README examples against the current build.

- `demos/div-plugin.html`: Quick Start example for the `div` auto-render plugin.
- `demos/api.html`: Basic API usage mirroring the README snippet.

Run a local server from the repo root, then open:

```
python3 -m http.server 8052
```

Visit:

- `http://localhost:8052/demos/div-plugin.html`
- `http://localhost:8052/demos/api.html`
